📂 折叠面板

什么是 MirrorWebView？

MirrorWebView 是一种在 AR 眼镜上实现双目 WebView 显示的方案。只创建一个 WebView 用于左眼，右眼通过 TextureView 实时镜像左眼画面。

焦点管理系统如何工作？

通过 JS 注入自动收集页面上所有可点击元素，支持原生标签、ARIA 属性、cursor:pointer 及各种框架事件绑定的检测。用户通过镜腿触摸板滑动切换焦点、单击触发点击。

暗色模式是如何实现的？

在 onPageFinished 时注入 JS 代码，遍历所有 DOM 元素将浅色背景改为黑色、深色文字改为灰色，并通过 MutationObserver 持续监听动态元素变化。

📋 使用步骤

  1. 在主页面选择要浏览的网站
  2. 等待页面加载完成（暗色模式+焦点系统自动注入）
  3. 向前滑动镜腿触摸板 → 焦点移到下一个元素
  4. 向后滑动镜腿触摸板 → 焦点移到上一个元素
  5. 单击触摸板 → 点击当前焦点元素
  6. 双击触摸板 → 返回上一页或退出
